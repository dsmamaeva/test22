Главная Каталог Высокотемпературные печи

ВЫСОКОТЕМПЕРАТУРНЫЕ ПЕЧИ

Серия напольных высокотемпературных печей с рабочим объемом камеры от 16 до 276 литров. Модульное исполнение позволяет быстро заменить вышедший из строя элемент печи: нагреватели, рабочую камеру, дверь. Система управления смонтирована в отдельном шкафу и располагается на боковой стенке корпуса. В качестве изоляции применяются специальные волокнистые материалы из оксида алюминия которые собираются с применением армирующих элементов для увеличения срока службы печи.

ОСНОВНЫЕ ВОЗМОЖНОСТИ:

  • Корпус печи с вентиляцией стенок для обеспечения малой температуры боковых поверхностей
  • Многослойная изоляция рабочей камеры с армированным сводом и боковыми стенками
  • Нагреватели из дисилицида молибдена
  • Вентиляционное отверстие в своде печи с ручным шибером
  • Плоскопараллельное открытие двери вбок
  • Программное управление технологическим процессом обжига
  • Независимый контроль температуры для аварийного отключения нагрева в случае превышения максимальной допустимой температуры
  • Архивирование технологических данных
